size_t Node::peek ( const size_t  codepoint) const

peek O(1) lookup around which this class is focused.

Lookup index associated with table (0 means error)

Parameters
codepoint
Returns
assoc

A codepoint is cut into 7 3bit chunks which are used to dereference the table 7 times after which the last dereference produces the associated value. The number 7 and 3 bits is chosen because 7*3 = 21 where 21 bits is the number of bits needed for Unicode codepoints up to the value 0x10FFFF (max). It also is parsimonious of space enabling a compromise between speed and size. This is a form of sparse table implementation.
